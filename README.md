# Setup
look for any setting in your browser that allows you to set the homepage as a custom url and set it to https://home-launcher.vercel.app
bonus: if you want it to also open in new tabs you can use [this extension](https://addons.mozilla.org/en-US/firefox/addon/new-tab-override/)
